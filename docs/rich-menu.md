# LINE リッチメニュー設定ガイド（T4.7）

LINE アプリのチャット画面下部に常設ボタンを表示する設定手順。
頻繁に使う記録ボタン（母乳・ミルク・うんち等）をワンタップで送れるようにする。

---

## 完成イメージ

```
┌──────────────┬──────────────┐
│     母乳     │    ミルク    │
├──────────────┼──────────────┤
│    うんち    │   おしっこ   │
├──────────────┼──────────────┤
│     寝た     │   操作一覧   │
└──────────────┴──────────────┘
```

タップすると、ラベルと同じテキスト（「母乳」「ミルク」等）が自動送信される。

---

## 設定方法（OA Manager ブラウザ画面）

画像の用意不要・コマンド操作不要でブラウザだけで完結する。

### 1. OA Manager を開く

[https://manager.line.biz/](https://manager.line.biz/) にアクセスし、koto-log のアカウントでログイン。

### 2. リッチメニュー画面へ移動

左サイドメニュー →「**トークルーム管理**」→「**リッチメニュー**」→ 右上の「**作成**」をクリック。

### 3. タイトルと表示期間を設定

| 項目 | 設定値 |
|---|---|
| タイトル | `kotolog-main`（管理用。ユーザーには見えない） |
| 表示期間 | 開始: 今日の日付 / 終了: 空欄（無期限） |
| メニューバーのテキスト | `メニュー` |
| デフォルトの表示 | **表示する**（チャットを開いたときに自動で開く） |

### 4. テンプレートを選択

「**テンプレートを選択**」から **大サイズ・6分割（2列×3行）** を選ぶ。

> テンプレートの見た目が「2×3グリッド」になっているものを選べばOK。

### 5. 画像を設定（内蔵エディタで作成）

「**画像を編集して適用**」をクリックすると内蔵エディタが開く。

外部ツールで画像を作らなくてよい。エディタ内で：

1. 背景色を選ぶ（白や薄いグレーなど）
2. 「**テキストを追加**」で各マスにラベルを入力する
3. フォントサイズ・色を調整して見やすくする
4. 「**適用**」で確定

> 凝ったデザインは不要。文字が読めれば十分機能する。

### 6. 各ボタンのアクションを設定

テンプレートの各エリア（6マス）をクリックし、それぞれ設定する。

**アクションの種別:** 「**キーワード送信**」を選ぶ（ユーザーがタップするとそのテキストが自動送信される）

| エリア | アクションラベル | キーワード（送信テキスト） |
|---|---|---|
| 左上 | `母乳` | `母乳` |
| 右上 | `ミルク` | `ミルク` |
| 左中 | `うんち` | `うんち` |
| 右中 | `おしっこ` | `おしっこ` |
| 左下 | `寝た` | `寝た` |
| 右下 | `操作一覧` | `操作一覧` |

### 7. 保存・公開

「**保存**」をクリック → 即時公開される。

「下書き保存」にすると未公開のまま保存される（後で「公開」に変更可能）。

---

## 確認方法

1. スマートフォンの LINE アプリで koto-log との **トーク画面** を開く
2. 画面下部に「**メニュー**」バーが表示されていれば成功
3. バーをタップするとリッチメニューが展開する
4. 各ボタンをタップしてテキストが送信され、ボットが応答することを確認

> **注意:** リッチメニューは **iOS / Android の LINE アプリのみ** 表示される。
> LINE for PC（Windows / macOS）では表示されない。

---

## うまくいかないとき

| 症状 | 確認ポイント |
|---|---|
| メニューバーが表示されない | 「デフォルトの表示」が「表示する」になっているか確認 / アプリを再起動 |
| ボタンをタップしても送信されない | アクション種別が「キーワード送信」になっているか確認 |
| 6分割テンプレートが見当たらない | 「大サイズ」を選択しているか確認（小サイズは3分割まで） |
| 保存できない | 全エリアにアクションが設定されているか確認（未設定のマスがあるとエラーになる場合がある） |

---

## 既存メニューの編集・削除

OA Manager のリッチメニュー一覧から、該当メニューの「**編集**」「**削除**」をクリックする。

削除前に「**公開終了**」にしておくと安全。

---

## 応用: 寝た / 起きた トグル（上級）

現構成では「寝た」ボタンのみで「起きた」はテキスト入力が必要。
睡眠状態に応じてボタンを自動切替したい場合、OA Manager では対応できない。
Messaging API の `richmenuswitch` アクション（REST API）と、webhook.py へのポストバックハンドラー追加が必要になる。

→ 出産後の実際の使い勝手を見てから検討する。

---

## API 方式（上級・参考）

OA Manager が使えない場合や自動化したい場合は curl コマンドで設定できる。
`LINE_CHANNEL_ACCESS_TOKEN` が必要。

```bash
# ① メニュー作成
curl -X POST https://api.line.me/v2/bot/richmenu \
  -H "Authorization: Bearer {TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{
    "size": {"width": 2500, "height": 1686},
    "selected": true,
    "name": "kotolog-main",
    "chatBarText": "メニュー",
    "areas": [
      {"bounds": {"x":    0, "y":    0, "width": 1250, "height": 562}, "action": {"type": "message", "label": "母乳",     "text": "母乳"}},
      {"bounds": {"x": 1250, "y":    0, "width": 1250, "height": 562}, "action": {"type": "message", "label": "ミルク",   "text": "ミルク"}},
      {"bounds": {"x":    0, "y":  562, "width": 1250, "height": 562}, "action": {"type": "message", "label": "うんち",   "text": "うんち"}},
      {"bounds": {"x": 1250, "y":  562, "width": 1250, "height": 562}, "action": {"type": "message", "label": "おしっこ", "text": "おしっこ"}},
      {"bounds": {"x":    0, "y": 1124, "width": 1250, "height": 562}, "action": {"type": "message", "label": "寝た",     "text": "寝た"}},
      {"bounds": {"x": 1250, "y": 1124, "width": 1250, "height": 562}, "action": {"type": "message", "label": "操作一覧", "text": "操作一覧"}}
    ]
  }'
# → {"richMenuId": "richmenu-xxx"} を控える

# ② 画像アップロード（2500×1686px の PNG）
curl -X POST "https://api-data.line.me/v2/bot/richmenu/{richMenuId}/content" \
  -H "Authorization: Bearer {TOKEN}" \
  -H "Content-Type: image/png" \
  -T richmenu.png

# ③ デフォルトに設定
curl -X POST "https://api.line.me/v2/bot/user/all/richmenu/{richMenuId}" \
  -H "Authorization: Bearer {TOKEN}"
```
